# Writing Guide

* 패러미터, 매개변수     -> 파라미터
* 한글English           -> 한글*English*
* English(English), English (English) -> English (*English*)
* 파이썬, python, 파이선 -> Python
* k-평균, k-means       -> k-Means
* 결측치                -> 결측값
* 열, column            -> 컬럼
