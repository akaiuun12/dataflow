웹은 HTML, CSS, JS로 이루어져 있다. 내 블로그도 예외는 아니다. 블로그를 원하는대로 뜯어고치려면 웹에 관한 지식이 필요하다. 다행히 HTML이나 CSS는 배우는데 그렇게 어렵지는 않다. 

여기는 블로그를 하면서 필요한 HTML 관련 지식을 정리한다.

## HTML Template
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
`<head>` 및 `<body>` 섹션은 HTML(Hypertext Markup Language) 문서의 기본 요소로서 서로 다른 목적을 가지며 웹 콘텐츠를 구조화하고 표시한다.

### `<head>`
`<head>` 섹션은 HTML 문서 상단에 위치한다.

웹 페이지에 대한 메타데이터와 정보를 포함하며, 일반적으로 사용자에게는 보이지 않는다. `<head>` 섹션 내의 요소는 웹 페이지의 가시적인 콘텐츠에 영향을 미치지 않는다. 하지만 기능성과 검색 엔진 최적화(SEO)에 중요한 역할을 한다. 검색 엔진, 브라우저 및 기타 서비스가 웹 페이지를 올바르게 이해하고 표시하는 데 도움을 준다.

`<head>` 섹션에서 발견되는 일반적인 요소는 다음과 같다:

* `<title>`: 웹 페이지의 제목을 설정하며 브라우저의 제목 표시줄이나 탭에 표시.
* `<meta>`: 문서에 대한 메타데이터. 문자 인코딩, 저자 및 설명과 같은 정보 표시.
* `<style>`: 웹 페이지의 스타일링을 위한 내부 CSS 포함.
* `<script>`: JavaScript 코드를 정의하거나 외부 스크립트 파일을 참조.
* `<link>`: 스타일 시트(CSS) 또는 아이콘 파일과 같은 외부 리소스를 지정.

```html
<!DOCTYPE html>
<html>
<head>
    <title>내 웹 페이지</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- 웹 페이지의 가시적인 콘텐츠는 여기에 위치합니다 -->
</body>
</html>
```

### `<body>`

`<body>` 섹션은 웹 페이지의 가시적인 콘텐츠를 배치하는 곳이다. 사용자가 웹 브라우저에서 볼 수 있는 텍스트, 이미지, 링크, 양식 등을 포함한다. 

`<body>` 섹션에서 사용되는 일반적인 HTML 요소로는 제목(`<h1>`, `<h2>` 등), 단락(`<p>`), 이미지(`<img>`), 링크(`<a>`), 목록(`<ul>`, `<ol>`, `<li>`) 등이 있다. 웹 페이지에 상호 작용성을 추가하기 위해 JavaScript 코드를 포함하거나 레이아웃과 디자인을 제어하기 위해 CSS 스타일을 적용하기도 한다.

```html
<!DOCTYPE html>
<html>
<head>
    <title>내 웹 페이지</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>내 웹 페이지에 오신 것을 환영합니다</h1>
    <p>이것이 페이지의 주요 콘텐츠입니다.</p>
    <img src="image.jpg" alt="예시 이미지">
    <a href="https://www.example.com">Example.com 방문하기</a>
</body>
</html>
```

#### `<table>`: 표 만들기  
```html
<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
```

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

HTML에서 테이블을 만들기 위해서는 `<table>` 태그를 사용한다. `<table>`로 감싸고 내부에 값을 입력하는 방식이다.

`<tr>`로 테이블의 행*row*을 만들 수 있다. 위 코드에서는 `<tr>` 태그를 두 번 사용하여 두 개의 행을 만들었다. `<th>`는 첫 번째 행 헤더를 의미한다. 헤더 이외는 `<td>`를 사용하여 값을 입력한다.

비슷하게 `<thead>`, `<tbody>`, `<tfoot>`은 하나의 표를 각 영역(header, body, footer)로 나눈다고 한다.


### <main> 태그
```html
<main>
    <h1>제목</h1>
    <p>내용</p>
</main>
```

> `<main>` 요소의 콘텐츠는 문서의 유일한 내용이어야 합니다. 사이드바, 탐색 링크, 저작권 정보, 사이트 로고, 검색 폼 등 여러 문서에 걸쳐 반복되는 콘텐츠는 포함해선 안됩니다. 그러나 검색 폼이 페이지의 주요 기능이라면 예외로 둘 수 있습니다.
> 
> `<main>`은 요소 개요에 영향을 주지 않습니다. `<body>` 등의 요소나 `<h2>`와 같은 제목 요소와 달리 `<main>`은 페이지의 개념적 구조를 바꾸지 않으며 온전히 정보 제공용입니다.
>
> From [MDN web docs](https://developer.mozilla.org/ko/docs/Web/HTML/Element/main)

`<main>`은 페이지의 주요 콘텐츠를 담는다. 특정한 기능을 수행하기보단 검색엔진 등에 페이지의 주요 콘텐츠를 알려주는 역할을 한다.

블로그 테마를 고치다가 `<section>`태그를 만났다. 어려운 얘기들이 써있는데 결과적으로 `<section>`의 기능을 구현한 웹 브라우저는 없으니 대신 `<article>`을 쓰는게 낫다는 것이 결론. 나중에 다시 읽어봐야겠다.


### HTML에 Font Awesome 5 적용하기
```html
...
<head>
    <!-- Font Awesome 5 -->
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.8.2/css/all.min.css"/>
</head>
...
```
Font Awesome은 웹 폰트처럼 웹 아이콘을 사용할 수 있게 해주는 서비스다. `<head>`부분에 위처럼 한줄의 코드로 불러와서 사용한다. 사용법이 간단해서 많은 개발자들이 쓴다.

```html
<i class="fa fa-file-image-o" aria-hidden="true"></i>
```

<i class="fa fa-file-image-o" aria-hidden="true"></i>

사용법은 넣고자 하는 부분에 `<i>` 태그를 써서 넣으면 된다. `<i>` 태그가 원래 아이콘용으로 만들어진 것은 아니지만 현재는 아이콘용으로 자주 사용된다. `class`를 바꾸면 아이콘의 종류를 바꿀 수 있고 시각적 용도로만 사용할거면 `aria-hidden="true"`로 해주는게 좋다. 

더 많은 아이콘은 [Font Awesome 홈페이지](https://fontawesome.com/v4.7.0/)에서 직접 찾아볼 수 있다.
